<b>Spring Cloud</b>
- Node.js 크롤러 서버가 하나의 Base Url만을 이용하여 개발 서버와 릴리즈 서버 모두에게 요청을 보낼 수 있도록 해서 편의성을 늘렸습니다.
- 개발 편의성을 위해 Feign 클라이언트를 사용해 외부 API를 호출 했습니다.

<b>QueryDSL</b>
- 컴파일 시점 문법 검사와 개발 편의성을 위해 QueryDSL을 사용했습니다.

<b>Redis</b>
- Refresh Token 관리를 위해 Redis를 사용했습니다.
- API 응답 캐시로 사용했습니다.
